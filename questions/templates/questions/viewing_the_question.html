Breaking Sticks

Hard 20 40
Algorithms
Input Format

The first line contains one integer which is the number of chocolate sticks. The second line contains space-separated integers , the lengths of the chocolate sticks.

Constraints

1< n < 100

Output Format

Print a single integer which is the maximum number of moves you can perform.

Sample Input 0

1 6

Sample Output 0

10

Explanation

You break a chocolate stick into three equal parts (1 move), then break each of them in half (3 moves), and then eat all six sticks (6 moves). This gives you 10 moves.
